§ 01

簡介

這是內文段落的預設樣式。使用 IBM Plex Sans 作為內文字型，搭配 IBM Plex Mono 處理程式碼與標籤。

修改 --accent 這一個 CSS 變數即可切換整體色系。所有元件的顏色都從這個變數衍生。

§ 02

使用方式

從 template 開始

  1. 複製 template.html，重新命名
  2. 修改 <title> 與 .doc-title
  3. 更新 sidebar 的 .nav-item 清單與對應的 id
  4. 依需求填入內容區段

切換色系

在 style-framework.css 最頂端找到 --accent，替換成你的品牌色：

:root {
  --accent: #1D9E75;   /* ← 改這一行 */
}
